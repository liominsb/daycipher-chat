DayCipher Chat 是一款基于**每日动态密钥**的安全加密聊天应用。它采用 Go 语言开发，结合现代 GUI 界面与AES-256 加密技术，为您提供"每日一变"的加密通信体验。

## 🚀 特性

### 🔐 安全加密
- **每日自动轮换密钥**：基于日期生成唯一加密密钥
- **AES-256-CBC 加密**：加密标准
- **前向安全性**：防止历史消息被破解
- **端到端加密**：消息仅在客户端解密

### 💻 技术优势  
- **纯 Go 实现**：无外部依赖，单文件可执行
- **跨平台 GUI**：基于 Fyne 框架
- **高性能**：低内存占用，高并发处理
- **开源透明**：代码完全开源，可审计

### 🎨 用户友好
- **现代化界面**：Material Design 风格
- **简单易用**：无需复杂配置
- **实时通信**：低延迟消息传递
- **主题切换**：支持亮色/暗色模式
